## 项目上下文记录

### 技术决策记录
- **代码编辑器选择**：使用CodeMirror + Monokai主题，支持Python语法高亮
- **后端架构**：使用Python HTTPServer，简单但功能完整
- **代码执行方式**：subprocess + 临时文件，确保安全性
- **前端框架**：原生HTML + CSS + JavaScript，避免复杂依赖

### 重要问题和解决方案
- **代码块显示问题**：字符重叠问题通过添加viewportMargin和refresh()解决
- **服务器错误处理**：ConnectionAbortedError通过优雅处理避免崩溃
- **Unicode编码问题**：使用英文错误消息避免latin-1编码错误
- **CORS支持**：添加OPTIONS请求处理和CORS头

### 用户偏好记录
- **代码编辑器大小**：用户反馈编辑器太小，已设置为400px最小高度，支持垂直调整
- **代码执行提示**：用户觉得"Connected to"提示不友好，已改为更简洁的输出
- **交互体验**：注重代码可读性和编辑便利性
